My participation in the [Kaggle LLM Agentic Legal Information Retrieval competition](https://www.kaggle.com/competitions/llm-agentic-legal-information-retrieval).
The goal is to build an agentic LLM pipeline that retrieves relevant Swiss legal citations for a given legal query, evaluated on citation-level F1.
This repo tracks my Kaggle notebook, experiment results, and learnings as I go. See DIARY.md for the full progress log.
